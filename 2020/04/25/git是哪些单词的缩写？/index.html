是哪些单词的缩写？

git 是一个开源的分布式版本控制系统，用以有效、高速的处理从很小到非常大的项目版本管理。git 原本是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的。

那么 git 这个名字是怎么来的呢？ 在 Linus 开发 git 的时候，原本有一个软件已经叫 git(GNU Interactive Tools)，所以 Linus 的这个分布式版本控制工具就只能叫 git-core。但后来可能是因为 git-core 更有地位，原本的 git 就改名为 gnuit，而 git-core 正式更名为 git。

说到这里，就自然而然的产生一个问题，git 究竟是哪些单词的缩写？比如，我们经常用的 Linux 命令 ls，它是 list files 的缩写；而 cd 则是 change directory 的缩写等等。那么 git 是否也是某些单词的缩写形式呢？

根据 Linus Torvalds 的 Initial revision of “git”，地址如下：
https://github.com/git/git/blob/e83c5163316f89bfbde7d9ab23ca2e25604af290/README

GIT - the stupid content tracker

“git” can mean anything, depending on your mood.

  • random three-letter combination that is pronounceable, and not
    actually used by any common UNIX command. The fact that it is a
    mispronounciation of “get” may or may not be relevant.
  • stupid. contemptible and despicable. simple. Take your pick from the
    dictionary of slang.
  • “global information tracker”: you’re in a good mood, and it actually
    works for you. Angels sing, and a light suddenly fills the room.
  • “goddamn idiotic truckload of sh*t”: when it breaks

原来 git 有多种含义，并非特定单词的缩写。如果一定要说 git 是某些单词的缩写，根据上面 Linus 给出的多种选项，显然 global information tracker（全局信息追踪器）这个选项比较靠谱。毕竟 git 的主要功能就是跟踪文本文件的修改。
